# 练习 1：第一次提交

## 目标
学习如何修改文件并提交到 GitHub

## 步骤

### 1. 修改这个文件
在下面添加你的名字：
- 我的名字是：___日落________

### 2. 查看更改
```bash
git status
git diff
```

### 3. 添加更改到暂存区
```bash
git add exercises/exercise-1.md
```

### 4. 提交更改
```bash
git commit -m "完成练习1：添加我的名字"
```

### 5. 推送到 GitHub
```bash
git push origin main
```

## 完成
如果你能在 GitHub 网页上看到你的名字，恭喜你完成了第一个练习！
