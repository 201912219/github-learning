# GitHub 学习教程

欢迎来到 GitHub 学习仓库！这里将帮助你从零开始掌握 GitHub。

## 什么是 GitHub？

GitHub 是一个**代码托管平台**，让你可以：
- 保存和管理代码
- 与他人协作开发
- 参与开源项目
- 展示你的技术能力

## 快速开始

```bash
# 克隆这个仓库
git clone https://github.com/201912219/github-learning.git

# 进入目录
cd github-learning

# 查看状态
git status
```
